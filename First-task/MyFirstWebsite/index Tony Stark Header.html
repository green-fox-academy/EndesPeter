Technical Terms

Logic gates are used to execute Boolean functions. It performs a logical operation on one or more binary inputs and produces a single binary output. ASCII is the internationally used computer language standard that translates the binary code/numbers into characters. RAM is the memory built into computers that enables the device to store information even after it is switched off. JAVA is one of the most used formal language that specifies a set of instructions that can be used to produce various kinds of output. DNS stands for Domain Name System.
